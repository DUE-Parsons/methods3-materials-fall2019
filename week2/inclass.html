In-class Exercises

Part 1: Styling with QGIS

  1. Download this data.
  2. Open the Council District Boundaries in QGIS.
  3. Open the Citibike Stations in QGIS.
  4. Experiment with styling both polygons and points. Label the council districts.
  5. Create a PDF of your map using a Print Layout.

Part 2: Project Projections

  1. Continue working with the map from Part 1.
  2. Go to Project > Project Properties.
  3. Select the CRS tab if it's not already selected.
  4. In the Filter text field, type 4326 and select WGS 84.
  5. Press OK.
  6. Try it again with 2263 and another projection or two.
